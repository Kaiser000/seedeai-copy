# 海报生成第一轮回归清单（固定坏例子）

> 目的：每次改动后都跑同一组例子，判断是否真的变好。

## Case 1：三图海报错位

- 名称：`three-images-mapping`
- 输入建议：
  - 主题：旅游/餐饮等明显需要 3 张图的海报
  - 尺寸：`1080x1920`
  - Prompt 示例：`做一张夏日旅行海报，包含头图、目的地实拍、优惠信息配图三块图片区`
- 检查点：
  - 头图区域显示的是头图，不是其它区块的图
  - 中间图片区和下方图片区不串图
  - 同一张图不会在多个区块重复出现（除非设计本来要求）
- 通过标准：3 张图都对应正确区块

## Case 2：Roll 后整页变化

- 名称：`roll-single-element-only`
- 输入建议：
  - 先生成任意完整海报
  - 只选中主标题文本
  - Roll 指令：`把标题改成更有冲击力的四字短句`
- 检查点：
  - 只有被选中的标题元素发生变化
  - 非目标元素（图片、按钮、背景块、其它文字）位置和样式不变
  - 画布不会被整页重画
- 通过标准：非目标元素变化数接近 0

## Case 3：同 Prompt 多次结果漂移

- 名称：`same-prompt-3-runs`
- 输入建议：
  - 固定同一条 Prompt 和尺寸
  - 连续生成 3 次
- 检查点：
  - 主结构是否一致（区块数量、主图区域、标题主次关系）
  - 不要求像素级一致，但不能每次版式完全换一个套路
- 通过标准：3 次结果的主结构一致性明显提升
