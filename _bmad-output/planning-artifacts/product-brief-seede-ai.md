# 产品简报：Seede AI 复刻版（海报生成器）

## 概要

本项目旨在复刻 Seede AI 的核心产品思路 — 一款 AI 驱动的海报设计工具。用户输入自然语言描述，AI 直接生成完整的海报设计，用户可以像做 PPT 一样拖拽、编辑每个元素，也可以对单个元素调用 AI 重新生成（Roll）。

核心技术路线：**LLM 生成 JSX + Tailwind CSS 代码 → 前端渲染为 DOM → 解析 DOM 树转换为 Fabric.js 画布对象 → 用户在画布上自由编辑 → 导出 PNG**。

当前阶段聚焦于**海报**这一单一场景，目标是跑通从"文字描述"到"可编辑海报"的完整流程，并打磨编辑器的交互体验。暂不考虑商业化。

## 要解决的问题

传统设计工具（Canva、稿定设计等）的核心模式是"选模板 → 改内容"，用户需要在大量模板中挑选，再逐一替换文字和图片。对于非设计专业的用户（运营、营销、中小企业主），这个过程依然繁琐且耗时。

AI 原生设计工具正在改变这个范式：**用户只说需求，AI 直接生成设计**。但当前市场上的 AI 设计工具大多只能输出扁平图片，用户无法精细编辑，只能"整体重新生成"或"从头再来"。

## 解决方案

采用"**代码即设计**"的技术思路：

1. **AI 生成结构化代码**（React JSX + Tailwind CSS），而非扁平图片
2. 前端将代码渲染为 DOM，再**解析为 Fabric.js 画布对象**，每个文本、形状、图片都是独立图层
3. 用户在画布上**自由拖拽、编辑文字、替换图片**
4. 对任意单个元素可以调用 AI **重新生成（Roll）**，无需重做整体设计
5. 编辑完成后一键**导出 PNG**

这使得 AI 生成的设计具备了传统设计工具的精细编辑能力。

## 差异化亮点

- **代码即设计**：LLM 生成的是结构化代码而非图片，天然支持多图层和精细编辑
- **元素级 AI 控制**：不是"全部重来"，而是对单个元素 Roll 一个新方案
- **零设计门槛**：自然语言输入，AI 完成布局、配色、排版的所有决策
- **聚焦海报场景**：MVP 不贪多，专注把海报做好

## 目标用户

需要制作海报但不具备专业设计能力的人群：社媒运营、营销人员、中小企业主、自由职业者。他们需要快速、好看、可微调的海报产出。

## 技术方案

| 层 | 技术选型 |
|---|---|
| 前端框架 | React + Tailwind CSS |
| 画布引擎 | Fabric.js |
| 后端 | Java（Spring Boot） |
| LLM | GLM-5（OpenAI 协议兼容，可配置切换） |
| 图像导出 | 浏览器端（Fabric.js canvas 导出） |

**核心流水线**：
```
用户输入 → LLM 需求分析（思考链）→ LLM 生成 JSX + Tailwind 代码
→ 前端编译渲染为 DOM → 解析 DOM 树 → 转换为 Fabric.js 画布对象
→ 用户编辑（拖拽/改文字/Roll 元素）→ 导出 PNG
```

## MVP 范围

**包含：**
- 自然语言输入生成海报
- JSX → DOM → Fabric.js 画布渲染
- 元素选中、拖拽、缩放
- 文字内容编辑
- 单元素 AI 重新生成（Roll）
- PNG 导出

**不包含（后续迭代）：**
- 多场景（PPT、社媒图文、长图等）
- 用户系统 / 历史记录
- 品牌配置
- 图像超分 / 背景移除等辅助工具
- 模板库
- 商业化功能（付费、订阅）

## 愿景

如果核心流程验证成功，后续可以：
- 扩展到更多设计场景（长图、社媒图文、PPT、电商详情页）
- 参考 Seede AI 的完整功能矩阵（参考图片输入、素材库、品牌一致性、批量生成）
- 接入更强的 LLM 提升生成质量
- 探索商业化路径
